# Riley's_Office — World Design

## World Premise
Riley's_Office frames the operating system as a headquarters campus. The world design is a comprehension layer that maps complex system state into memorable spaces.

## Spatial Metaphor
### Riley's_Office Campus
The global shell containing all projects, runtime infrastructure, inbox, and activity.

### Project Offices
Each project becomes an office with its own mission, health, active run, and team footprint.

### Agent Desks
Each active agent occupies a desk within a project office or shared operations room. Desks expose role, current contract, live status, artifacts, and conversation entry points.

### Shared Rooms
- **Mission Control** — global overview and alerts
- **Delivery Wall** — artifacts, milestones, QA, reviewer state
- **Ops Room** — runtime health and system signals
- **Inbox** — threaded interventions and direct chats
- **Archive Cabinet** — historical evidence and prior runs

## Design Principles
- The metaphor must reduce cognitive load, not add fiction.
- Every spatial object corresponds to a real domain object.
- Navigation depth should remain shallow.
- Distinctive rooms help memory and speed.

## Environmental Language
- Clean, dark, premium command-center aesthetic
- Glass/steel/card surfaces rather than cartoon office art
- Subtle depth layers and lighting zones
- Project identity expressed through accent color and insignia, not full theme rebuilds

## State Encoding
- Desk glow / badge conveys agent state
- Room pulse conveys live activity intensity
- Office aura conveys project health and urgency
- Delivery wall markers show milestone and quality gate state

## World Map Structure
1. Office Lobby
2. Project Corridor
3. Project Office
4. Room View
5. Desk Detail

## MVP World Design Constraint
Use 2D/2.5D layered interfaces only. No free-roam avatar movement. Spatiality comes from composition, transitions, and map semantics.

## Future Expansion
- Multi-floor offices for larger projects
- Multi-user presence within rooms
- Voice/ambient event cues
- Time-travel playback of historical runs
- Personalized desk layouts per agent role
